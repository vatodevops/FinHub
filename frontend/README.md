# Frontend

Panel web para:
- overview consolidado
- cuentas
- transacciones
- gasto mensual
- revisión de duplicados
- pagos recurrentes y calendario
- gastos manuales planificados

## Arranque local

```bash
cd projects/finhub/frontend
npm install
npm run dev
```

Por defecto espera la API en `http://localhost:8081/api`.

## Cómo verla en localhost

- Frontend: <http://localhost:3001>
- Backend API: <http://localhost:8081/api>
- Health: <http://localhost:8081/api/health>
- Overview: <http://localhost:8081/api/overview>
